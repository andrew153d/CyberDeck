# CyberDeck
A CyberDeck containing a SDR, 5G conenctivity, and other goodies

## Components ##
- Raspbery Pi
  - USBA 2.0
  - USBA 2.0
  - USBA 3.0
  - USBA 3.0
  - USBC : power
  - RJ45 : network
  - Micro HDMI : display
- BladeRF
  - USB 3.0 Type B
  - 5V Barrel Jack
  - 4x SMA
- Waveshare 5G HAT
  - Micro USB : power
  - USBA 3.0: network
- Screen
  - power: microusb
  - touch: microusb
  - display: hdmi
- Fan
